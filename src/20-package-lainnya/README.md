# Package Lainnya
- Sebenarnya masih ada beberapa package lainnya yang tidak dibahas.
- Hal ini dikarenakan package tersebut terlalu kompleks jika harus dibahas dalam 1 chapter, oleh karena itu package-package berikut akan dibahas di materi tersendiri, seperti: Package Context, Net, Testing, Template, Database, JSON dan Embed.
